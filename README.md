# 🧮 Premium Calculator

A modern and responsive Calculator developed using HTML, CSS and JavaScript as part of the CodeAlpha Frontend Development Internship.

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage (%)
- AC (Clear All)
- Delete Button
- Keyboard Support
- Responsive Design
- Glassmorphism UI
- Animated Background

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
